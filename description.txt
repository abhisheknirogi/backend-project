ğŸ›’ E-Commerce Backend API (Node.js + Express + MongoDB)

This project is a RESTful e-commerce backend API built using Node.js, Express, and MongoDB, following a clean separation of concerns with routes, controllers, models, and middleware.

The backend supports user authentication, role-based authorization, product management, cart handling, order processing, and payment workflows, making it suitable for real-world e-commerce applications.

Authentication is handled using JWT, with secure middleware to protect routes and restrict admin-only operations. The system is designed to be modular, scalable, and easily extendable.

This project serves as a production-style backend and a reference implementation that is later recreated using FastAPI (Python) to demonstrate backend framework and language flexibility.
